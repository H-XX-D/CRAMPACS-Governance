# CRAMPS Brand Kit

This folder defines how CRAMPS should look, sound, and organize practitioner-facing material.

Use it when creating:

- public descriptions
- training material
- printouts
- supervisor packets
- domain packs
- workbooks
- implementation decks
- platform screens

Core files:

- `CRAMPS_BRAND_SYSTEM.md`
- `CRAMPS_DOCUMENT_STYLE_GUIDE.md`
- `CRAMPS_MESSAGE_ARCHITECTURE.md`
