# CRAMPACS Brand Kit

This folder defines how CRAMPACS should look, sound, and organize practitioner-facing material.

Use it when creating:

- public descriptions
- training material
- printouts
- supervisor packets
- domain packs
- workbooks
- implementation decks
- platform screens

Core files:

- `CRAMPACS_BRAND_SYSTEM.md`
- `CRAMPACS_DOCUMENT_STYLE_GUIDE.md`
- `CRAMPACS_MESSAGE_ARCHITECTURE.md`
