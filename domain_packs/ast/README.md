# cramps-ast / CRAMPS-AST Starter Pack

**Domain:** Astronomy and astrophysics

This pack contains domain-specific printable templates for both:

- `cramps-ast` lightweight preflight
- `CRAMPS-AST` full assurance study

Use the lowercase documents for a one to two day triage pass. Use the uppercase documents only after creating a full protocol lock and importing the preflight package with review disposition.

## Files

- `cramps-ast_PREFLIGHT_SCOPE.md`
- `cramps-ast_PREFLIGHT_SOURCES.csv`
- `cramps-ast_PREFLIGHT_ROWS.csv`
- `cramps-ast_PREFLIGHT_GOTCHAS_PRINTABLE.md`
- `cramps-ast_PREFLIGHT_DECISION.md`
- `CRAMPS-AST_FULL_PROTOCOL_ADDENDUM.md`
- `CRAMPS-AST_RELEASE_GATE_PRINTABLE.md`
- `CRAMPS-AST_DOMAIN_GOVERNANCE_PRINTABLE.md`

## Domain Coordinates

sky coordinate, redshift, wavelength, period, phase, cadence, source class

## Domain Nulls

follow-up non-detections, survey fields with no event, searched spectral windows with no feature

## Main Gotcha

sky coverage bias, cadence bias, follow-up selection bias, calibration drift

## Standards Anchors

FITS, IVOA, VOEvent, NASA ADS, FAIR/DataCite
