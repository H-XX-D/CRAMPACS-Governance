# cramps-clim / CRAMPS-CLIM Starter Pack

**Domain:** Climate and Earth systems

This pack contains domain-specific printable templates for both:

- `cramps-clim` lightweight preflight
- `CRAMPS-CLIM` full assurance study

Use the lowercase documents for a one to two day triage pass. Use the uppercase documents only after creating a full protocol lock and importing the preflight package with review disposition.

## Files

- `cramps-clim_PREFLIGHT_SCOPE.md`
- `cramps-clim_PREFLIGHT_SOURCES.csv`
- `cramps-clim_PREFLIGHT_ROWS.csv`
- `cramps-clim_PREFLIGHT_GOTCHAS_PRINTABLE.md`
- `cramps-clim_PREFLIGHT_DECISION.md`
- `CRAMPS-CLIM_FULL_PROTOCOL_ADDENDUM.md`
- `CRAMPS-CLIM_RELEASE_GATE_PRINTABLE.md`
- `CRAMPS-CLIM_DOMAIN_GOVERNANCE_PRINTABLE.md`

## Domain Coordinates

latitude, longitude, depth, pressure level, season, climate mode, threshold, basin

## Domain Nulls

comparable regions without anomaly, model runs without recurrence, negative attribution studies

## Main Gotcha

spatial autocorrelation, temporal autocorrelation, non-stationary baseline, model-family dependence

## Standards Anchors

WMO, CF Conventions, CMIP/ESGF, IPCC uncertainty, FAIR
