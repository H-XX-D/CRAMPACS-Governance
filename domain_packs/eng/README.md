# cramps-eng / CRAMPS-ENG Starter Pack

**Domain:** Engineering reliability

This pack contains domain-specific printable templates for both:

- `cramps-eng` lightweight preflight
- `CRAMPS-ENG` full assurance study

Use the lowercase documents for a one to two day triage pass. Use the uppercase documents only after creating a full protocol lock and importing the preflight package with review disposition.

## Files

- `cramps-eng_PREFLIGHT_SCOPE.md`
- `cramps-eng_PREFLIGHT_SOURCES.csv`
- `cramps-eng_PREFLIGHT_ROWS.csv`
- `cramps-eng_PREFLIGHT_GOTCHAS_PRINTABLE.md`
- `cramps-eng_PREFLIGHT_DECISION.md`
- `CRAMPS-ENG_FULL_PROTOCOL_ADDENDUM.md`
- `CRAMPS-ENG_RELEASE_GATE_PRINTABLE.md`
- `CRAMPS-ENG_DOMAIN_GOVERNANCE_PRINTABLE.md`

load, vibration frequency, temperature, firmware version, supplier lot, cycle count, failure mode
