# CRAMPACS-MAT / crampacs-mat Domain Governance Printable

**Domain:** Materials science

## Assurance Split

- `crampacs-mat`: one to two day preflight.
- `CRAMPACS-MAT`: full assurance system after protocol lock.

## Coordinates

- composition ratio
- dopant level
- phase
- lattice parameter
- processing temperature
- operating condition

## Nulls and Non-Events

- failed syntheses
- tested materials without property jump
- simulations with no predicted anomaly

## Gotchas

- unreported failed syntheses
- hidden processing parameters
- batch variation
- simulation convergence artifacts

## Standards Anchors

- OPTIMADE
- NOMAD/FAIR-DI
- Materials Project provenance
- PIF/GEMD
- ISO 17025
