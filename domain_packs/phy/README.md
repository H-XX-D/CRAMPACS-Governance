# CRAMPACS-PHY / crampacs-phy Starter Pack

**Domain:** Physics and physical anomaly catalogs

This pack contains domain-specific printable templates for both:

- `crampacs-phy` lightweight preflight
- `CRAMPACS-PHY` full assurance study

Use the lowercase documents for a one to two day triage pass. Use the uppercase documents only after creating a full protocol lock and importing the preflight package with review disposition.

## Files

- `crampacs-phy_PREFLIGHT_SCOPE.md`
- `crampacs-phy_PREFLIGHT_SOURCES.csv`
- `crampacs-phy_PREFLIGHT_ROWS.csv`
- `crampacs-phy_PREFLIGHT_GOTCHAS_PRINTABLE.md`
- `crampacs-phy_PREFLIGHT_DECISION.md`
- `CRAMPACS-PHY_FULL_PROTOCOL_ADDENDUM.md`
- `CRAMPACS-PHY_RELEASE_GATE_PRINTABLE.md`
- `CRAMPACS-PHY_DOMAIN_GOVERNANCE_PRINTABLE.md`

## Domain Coordinates

mass, energy, frequency, redshift, coupling, cross section, decay channel, event topology

## Domain Nulls

null searches, exclusion contours, control regions, sidebands, non-detections, failed replications

## Main Gotcha

look-elsewhere effect, theory-fashion clustering, shared detector pipelines, plot digitization error
