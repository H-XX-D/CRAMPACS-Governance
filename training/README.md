# CRAMPACS Training Kit

This folder contains teaching material for introducing CRAMPACS to teams.

Use this kit to run:

- a 60 minute executive briefing
- a 2 hour supervisor orientation
- a 1 day `crampacs-*` preflight workshop
- a 3 day practitioner course
- a 5 module full-system course

Core files:

- `CRAMPACS_TRAINING_GUIDE.md`
- `INSTRUCTOR_GUIDE.md`
- `LEARNER_WORKBOOK.md`
- `EXERCISE_PACKETS.md`
- `COMPETENCY_RUBRIC.md`
- `SLIDE_OUTLINE.md`
